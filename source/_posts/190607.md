## 使用

### 获取稳定版本：

https://github.com/blleng/hexo-theme-lx/releases

解压后将文件夹命名为`lx`放在`themes`文件夹中。

### 获取具有最新特性的版本：

在博客根目录:

```bash
git clone https://github.com/blleng/hexo-theme-lx themes/lx
```

### 启用本主题：

更改站点设置文件`_config.yml`:

```yml
theme: lx
```

主题的配置将在`themes/lx/_config.yml`中进行。
